zhtokenizer: Chinese Word Tokenizer
=====

About
-----
This tool is used for tokenize Chinese sentence into words.

How to Compile?
-----
make

How to run?
-----
./zhtokenizer < chinese-text-in-utf8

Where does the dictionary come from?
